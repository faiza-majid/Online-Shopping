# Online Shopping Mini Project (PHP + MySQL)

## Setup (XAMPP)
1. Start Apache and MySQL.
2. Open **phpMyAdmin** → Import → select `db.sql` from this folder → Import.
3. Copy the entire `onlineshoppings` folder to `C:\xampp\htdocs\` (Windows) or `/Applications/XAMPP/htdocs/` (macOS).
4. Visit `http://localhost/onlineshoppings/onlineshopping/`.

## Features
- Customers & Products CRUD
- Create Orders with multiple items (up to 5 rows in the form)
- View Order with totals, paid and due
- Record Payments with validation
- Clean basic UI (no frameworks)

> Tip: If you change MySQL username/password, update them in `db.php`.
